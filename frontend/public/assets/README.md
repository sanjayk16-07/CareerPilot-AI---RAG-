# CareerPilot Branding Assets

Place the original, unmodified CareerPilot brand files in this directory:

- `logo.png`
- `icon.png`
- `favicon.ico`
- `app-icon-512.png`
- `logo-dark.png`
- `logo-light.png`

The app references these exact filenames from `/assets/*`. Do not crop, recolor, distort, regenerate, or rename the source artwork.
